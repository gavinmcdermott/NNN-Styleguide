Success

Description

We use to give feedback to people in the app during important form filling or if an error / success state occurred when a change was submitted.

- We simply apply the .notice--success class to the .notice element to get the success styling

- Note the use of the .active class within the element. Animation is built into the .notice class so that when the active class is applied, it will spring into view.

- We have large and small notification elements. We can use the large one by simply adding the .notice--large class to the notice element.

Large example

Large success! Hannah's recipe was successfully updated.

Small example

Success! Hannah's recipe was updated.

Code snippet

<!-- Large --> <div class="notice notice--large active notice--success"> <p class="nomargin"><strong>Large success! </strong> Hannah's recipe was successfully updated.</p> </div> <!-- Small --> <div class="notice active notice--success"> <p class="nomargin"><strong>Success! </strong> Hannah's recipe was updated.</p> </div>
