Font utilities

Here are the font utils we use to separate to keep things flexible.

  • CSS Class
    Result
  • .f--small
    makes font size 14px
  • .f--bold
    bolds text weight (fw 900)
  • .f--reg
    regular text weight (fw 400)
  • .f--dim
    sets text color to #8C9196
  • .f--warn
    sets text color to #EA4343
  • .align--center
    center aligns the text in the element
  • .el--dim
    sets the elements opacity to 75%
