Buttons

Description

Buttons come after form elements, and are typically wrapped in a .button-group class which handles padding and spacing.

- Buttons can either be a tags with the .button class applied, or a button tag with the same .button tag applied.

- The colors of the buttons are managed by the priorities used in the NNN app.

Example

Save Cancel

Code snippet

<div class="line"> <div class="lastUnit size1of1 button-group"> <a class="button pri0" href="">Save</a> <a class="button pri1" href="">Cancel</a> </div> </div>

Inputs

Text input description

Basic form text inputs used throughout the app

- Input fields are wrapped with the .field-wrap class.

Text input example

Text input code snippet

<div class="field-wrap"> <label>State</label> <input type="text" /> </div>

Text input with error

Basic form text field errors used throughout the app

- Input fields have the .invalid class added.

- Spans below the input hold the error message.

Text input error example

This is what a field error looks like.

Text input error code snippet

<div class="field-wrap"> <label>Last name</label> <input class="invalid" type="text" value="some bad value" /> </div> <span class="invalid-error"> This is what a field error looks like. </span>

Selects

Description

Selects are used when selecting a bunch of possible options. We use the native element for better access on mobile devices.

Example

Code snippet

<div class="field-wrap"> <label for="">State</label> <div class="select-wrap"> <select> <option value="volvo">CA</option> <option value="saab">NE</option> <option value="mercedes">WI</option> </select> <span class="icon fa fa-chevron-down"></span> </div> </div>

Radios

No-descriptive-text description

Radios are used when choosing between mutually exclusive options in the app.

No-descriptive-text example

No-descriptive-text code snippet

<div class="field-wrap radio-wrap"> <input checked="true" name="h1" type="radio" /> <label for="">My door is easy to access</label> </div>

With-descriptive-text description

This radio can also have a nice little description with it :)

With-descriptive-text example

Some title

With-descriptive-text code snippet

<div class="field-wrap radio-wrap"> <input checked="true" name="h1" type="radio" /> <label for="">My door is easy to access</label> <div class="radio-content radio-content--reveal"> <p class="">Some title</p> <textarea class="nomargin" name="" id="" rows="3"></textarea> </div> </div>

Checkboxes

Hidden-descriptive-text description

Checkboxes are pretty self explanatory...

Hidden-descriptive-text example

Hidden-descriptive-text code snippet

<div class="field-wrap"> <label class="checkbox"> <input type="checkbox" checked class="checkbox__option-input" /> <span>Feed Hannah 2 meals of NomNomNow each day</span> <div class="checkbox-content checkbox-content--reveal"> <p class="nomargin">Note how this info shows up when the box is checked.</p> </div> </label> </div>

With-descriptive-text description

Checkboxes can also have descriptions.

- We add the .checkbox-content--reveal class to the checkbox content div

With-descriptive-text example

With-descriptive-text code snippet

<div class="field-wrap"> <label class="checkbox"> <input type="checkbox" checked class="checkbox__option-input" /> <span>Feed Hannah 2 meals of NomNomNow each day</span> <div class="checkbox-content"> <p class="nomargin">Note how this info shows up when the box is checked.</p> </div> </label> </div>

Regarding update notices

When someone is updating a form and we're giving them feedback as to what they're changing as they go (because we're assuming it's an important form), we'll show these types of notifiers as they make changes.

- These notices are within the field-wrap div.

With-notice-text example

Pending meal change: Hannah was previously receiving a half order.

With-notice-text code snippet

<!-- The field --> <div class="field-wrap"> <label class="checkbox"> <input type="checkbox" checked class="checkbox__option-input" /> <span>Feed Hannah 2 meals of NomNomNow each day</span> </label> </div> <!-- The notification --> <div class="notice active notice--success"> <p class="nomargin"><strong>Pending meal change:</strong> Hannah was previously receiving a half order.</p> </div>
