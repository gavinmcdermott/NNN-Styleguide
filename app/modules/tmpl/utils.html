Spacing

Spacing

Spacing utility classes deal with padding and margin applied to elements in order to maintain consistent style and rhythm (vertical and horizontal) in the app.

  • Utility Tag
    Result
  • .center--h
    centers element horizontally in parent (if parent is relative)
  • .center--v
    centers element vertically in parent (if parent is relative)
  • .selfclear
    clears all floats after the element
  • .float--left
    floats element to left
  • .float--right
    floats element to right
  • .pad--lev6
    adds padding of 42px to the element
  • .below--*
    adds a margin of *px to the element (based in multiples of 6)
  • .pad--b*
    adds a bottom padding of *px to the element (based in multiples of 6)
  • .pad--0
    kills padding of the element
  • .pad--l0
    kills left padding of the element
  • .pad--r0
    kills right padding of the element
  • .pad--tb0
    kills top and bottom padding of the element

Positioning

Positioning

Positioning utility classes deal with the actual position of the box element within the markup of the page.

  • Utility Tag
    Result
  • .center--h
    centers element horizontally in parent (if parent is relative)
  • .center--v
    centers element vertically in parent (if parent is relative)
  • .selfclear
    clears all floats after the element
  • .float--left
    floats element to left
  • .float--right
    floats element to right
  • .nofloat
    kills float of element
  • .nomargin
    kills margin of element
  • .inline
    makes element an inline element
  • .invisible
    makes element element invisible
  • .visible
    makes element element visible

Colors

Colors

Coloring deals with filling the background of elements.

  • Utility Tag
    Result
  • .fill--lightgray
    sets background to #E0E0E0
  • .fill--success
    sets background to #45CEBD and text to #FFF
