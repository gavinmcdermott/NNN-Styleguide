Tile list

Description

We use tiles to give people a way to interact with items that have a lot of potential stats or actions to take within them -- from within a list format. Additionally, tiles are used present information in a way that allows users to scan / confirm information quickly.

Note: You can also increase the padding (e.g.: for easier scanability in a confirmation list scenario) using the .tile-list--large class on the .tile-list wrapper.

Example

  • Hannah's Meals

    Edit

    Eating 2 cups of Tasty Turkey Fare twice daily

  • Delivery Address

    Edit

    Delivering to 899 Pine Street, #801 San Francisco, CA 94108

Code snippet

<ul class="tile-list"> <li class="tile"> <div class="tile__image"> <div class="icon icon--large fa fa-check"></div> </div> <div class="tile__body"> <div class="line"> <h4 class="float--left">Hannah's Meals</h4> <span class="float--right"><a>Edit</a></span> </div> <p>Eating 2 cups of Tasty Turkey Fare twice daily</p> </div> </li> </ul>

Standard list

Description

Standard lists are used to show a list of vertically aligned nav links.

Example

  • Something
  • Something else
  • Something other

Code snippet

<ul class="standard-list"> <li class="selected"><a href="">Something</a></li> <li><a href="">Something else</a></li> <li><a href="">Something other</a></li> </ul>

Bordered list

Description

Bordered lists are used to call out lists of facts or stats in a table-like format. They help with scanability of data. Typically they are in a 2 column format with a label in the left column, and value in the right column.

Example

  • Dog name
    Wutang Pup
  • Favorite toy
    Ball of some kind

Code snippet

<ul class="bordered-list"> <li class="size1of1 lastGroup"> <div class="unit size1of2"><strong>Dog name</strong></div> <div class="lastUnit size1of2">Wutang Pup</div> </li> <li class="size1of1 lastGroup"> <div class="unit size1of2"><strong>Favorite toy</strong></div> <div class="lastUnit size1of2">Ball of some kind</div> </li> </ul>

Stats list

Description

Stats lists are used to display important or key pieces of information to a customer. Typically they have a label on the left leading to the statistic on the right.

Example

  • Size 2 and 3/4 cups
  • Shape Square-ish

Code snippet

<ul class="stats-list"> <li class="size1of1 lastGroup"> <span class="f--reg f--small">Size</span> <span class="float--right">2 and 3/4 cups</span> </li> <li class="size1of1 lastGroup"> <span class="f--reg f--small">Shape</span> <span class="float--right">Square-ish</span> </li> </ul>

Linear list

Description

Linear lists typically how we show a list of horizontally aligned links or data.

Example

  • Something
  • Another
  • Oncemore

Code snippet

<ul class="linear-list"> <li><a href="">Something</a></li> <li><a href="">Another</a></li> <li><a href="">Oncemore</a></li> </ul>
