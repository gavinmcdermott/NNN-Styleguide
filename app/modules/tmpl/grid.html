Code snippet

When using the grid, wrap the columns using a line and use lastUnit for the last column. We're using Nicole Sullivan's OOCSS grid naming conventions.

This basic example below applies to all of the ratios we support: 1/1, 1/2, 1/3, 1/4, and 1/8.

<div class="line"> <div class="unit size1of3"></div> <div class="unit size1of3"></div> <div class="lastUnit size1of3"></div> </div>

Nesting and gutters

Basic nesting example

Notice that we want to eliminate the gutters on these nested grids, otherwise we'd have a 30px padding within each nested item. This odd behvior messes with readability, consistency and looks sloppy.

To avoid this, we use the group class to eliminate gutters on the parent element of the columns being nested.


- When possible, limit nesting grids to 2 levels deep.

- Also note the use of the last* prefix on the last unit or group used.
