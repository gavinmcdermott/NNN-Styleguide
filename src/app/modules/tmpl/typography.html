Here's the behavior you can expect from your grid on different screen sizes.

  • CSS Class
    Result
  • .f--bold
    bolds text weight (fw 900)
  • .f--reg
    regular text weight (fw 400)
  • .f--warn
    sets text color to #EA4343
  • .el--dim
    sets the elements opacity to 80%
